## PM2 
`pm2 start scr.app.js` 开启服务
`pm2 list` 						 服务列表
`pm2 stop id` 				 停止服务（id）

## Babel

`yarn add @babel/core @babel/cli @babel/preset-env --dev`
`yarn add @babel/node --dev`

- `@babel/core` Babel 进行代码转换的核心
- `@babel/cli` 内置 CLI 可通过命令行编译文件
- `@babel/preset-env` 预设机盒，允许使用最新的 ECMA 特性
- `@babel/node`
